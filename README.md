# YouTube App

---

#### Getting started

Clone the repo locally
`git clone https://github.com/omarbakeer/youtube-app.git`

Navigate to the directory and install npm_modules
`cd youtube-app`
`npm install`

To run the app
`REACT_APP_GOOGLE_API_KEY="**{YOUR APP KEY}**" npm start`

OR

use the **.env** file

To run test cases
`npm run test`
